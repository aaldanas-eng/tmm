# Thai Massage Manager

Sistema de gestión integral para centros de masaje tailandés.

## Características

- ✅ Gestión de usuarios y roles
- ✅ Registro de sesiones de trabajo
- ✅ Dashboard con estadísticas y gráficos
- ✅ Gestión de múltiples spas
- ✅ Cálculo automático de IVA
- ✅ Interfaz responsive

## Configuración

### 1. Base de Datos Supabase

1. Crear una cuenta en [Supabase](https://supabase.com)
2. Crear un nuevo proyecto
3. Ejecutar los scripts SQL proporcionados en la carpeta `database/`
4. Configurar las credenciales en `index.html`

### 2. Configuración Local

1. Clonar el repositorio
2. Abrir `index.html` en un navegador web
3. Configurar las credenciales de Supabase en el archivo
